# 配置
  `React` 可與編輯器、`TypeScript`、瀏覽器擴充功能和編譯器等多種工具集成。本節將幫助你完成環境的配置。

## 編輯器配置
  - ### 說明
    請參閱官方推薦的 [編輯器設置](/pages/f2e/docReact/reactDev/03/reactDev_03_01.html)，了解如何配置它們以與 `React` 協同工作。

## 使用 TypeScript
  - ### 說明
    `TypeScript` 是一種為 `JavaScript` 代碼庫添加類型定義的流行方式。可透過相關指南了解如何將 `TypeScript` 集成到你的 `React` 項目中。

## React 開發者工具
  - ### 說明
    `React` 開發者工具（`React Developer Tools`）是一款瀏覽器擴充功能，它能夠檢查 `React` 組件、編輯 `props` 和 `state`，並識別性能問題。

## React Compiler
  - ### 說明
    `React Compiler` 是一款自動優化 `React` 應用程序的工具。
