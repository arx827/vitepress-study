# App Router

歡迎來到 `Next.js` `Foundations` 課程！在這個免費的互動課程中，你將透過建構一個全端網頁應用程式來學習 `Next.js` 的主要功能。

## 我們將建構什麼
  ![儀表板專案的桌面版與行動版畫面截圖](/pages/f2e/docReact/nextDev/imgs/01/nextDev_01_01.avif)

  - 在這門課程中，我們將建構一個財務儀表板，其中包含:
    - 一個公開的首頁
    - 一個登入頁面
    - 需要透過驗證才能存取的儀表板頁面
    - 使用者可以新增、編輯、刪除發票的功能

  - 這個儀表板還會搭配一個資料庫，你將在 [稍後的章節](/pages/f2e/docReact/nextDev/01/nextDev_01_06) 中設定它。

  - 在課程結束時，你將具備開始建構全端 `Next.js` 應用程式所需的核心技能。

## 概覽
  - 以下是這門課程中你將學到的功能概覽：
    - **樣式設計(Styling)**：在 `Next.js` 中為應用程式設計樣式的各種方式。

    - **最佳化(Optimizations)**：如何最佳化圖片、連結與字型。

    - **路由(Routing)**：如何使用檔案系統路由建立巢狀版面配置與頁面。

    - **資料擷取(Data Fetching)**：如何在 `Vercel` 上設定 `Postgres` 資料庫，以及擷取與串流資料的最佳實務。

    - **搜尋與分頁(Search and Pagination)**：如何使用 `URL` 搜尋參數實作搜尋與分頁功能。

    - **資料異動(Mutating Data)**：如何使用 `React Server Actions` 異動資料，並重新驗證 `Next.js` 快取。

    - **錯誤處理(Error Handling)**：如何處理一般錯誤及 `404` 找不到頁面的錯誤。

    - **表單驗證與無障礙設計(Form Validation and Accessibility)**：如何進行伺服器端的表單驗證，以及提升無障礙性的技巧。

    - **驗證(Authentication)**：如何使用 [`NextAuth.js`](https://next-auth.js.org/) 與 `Proxy` 為應用程式加入身分驗證。

    - **中繼資料(Metadata)**：如何新增中繼資料，並為應用程式的社群分享做好準備。

## 先備知識
  - 這門課程假設你已具備 `React` 與 `JavaScript` 的基本理解。如果你剛接觸 `React`，建議先修習 [React 基礎](/pages/f2e/docReact/nextDev/nextDev_03) 課程，學習 `React` 的基礎概念，例如元件、props、state、hooks，以及較新的功能如 Server Components 與 Suspense。

## 系統需求
  - 在開始這門課程之前，請確認你的系統符合以下需求:
    - 已安裝` Node.js 20.9` 或更新版本。[點此下載](https://nodejs.org/en)。
    - 作業系統：`macOS`、`Windows`(含 WSL)或 `Linux`。

  - 此外，你還需要一個 [GitHub 帳號](https://github.com/join/) 以及一個 [Vercel 帳號](https://vercel.com/signup)。
