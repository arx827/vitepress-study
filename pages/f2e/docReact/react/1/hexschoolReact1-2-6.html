6. 物件傳參考特性 ​

打開檔案 /es6/objectReference.html ​

  • 1. 物件是以傳參考的形式賦值 ​

    ts
    const person = {
  name: '小明',
  obj: {}
}

const person2 = person;
person2.name = '杰倫';

console.log(person.name);
console.log(person === person2);    // true

    • 3-1 淺層拷貝 ​

      ts
      const person = {
  name: '小明',
  obj: {}
}

const person2 = { ...person };
person2.name = '杰倫';
person.2.obj.name = 'person2'
console.log(person, person2)

      const person2 = Object.assign({}, person)
